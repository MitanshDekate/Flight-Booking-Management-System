# Flight Booking Management System

A MySQL-based Flight Booking Management System designed to manage airlines, flights, passengers, and ticket bookings efficiently. The project uses relational database concepts, foreign keys, and triggers for automated logging and data integrity.

---

## Features
- Airline management
- Flight scheduling
- Passenger registration
- Ticket booking system
- Booking status tracking
- Automated activity logs using triggers

---

## Technologies Used
- MySQL
- HTML/CSS
- PHP

---

## Database Tables
- `airlines`
- `flights`
- `passengers`
- `bookings`
- `logs`

---

## Key Concepts
- Primary & Foreign Keys
- Triggers
- Relational Database Design
- Data Integrity Constraints

---

mysql -u root -p flightms < database.sql
